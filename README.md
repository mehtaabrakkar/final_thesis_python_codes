## Overview
This repository contains the Python code used in my thesis. 
The analysis was conducted using Jupyter Notebooks.

## Files
- `thesis - H1.ipynb` – initial script for H1 data analysis 
- `thesis - H1 final.ipynb` – final script for HI data analysis
- `thesis - H2.ipynb` – script for H2 epidemiological study coding and analysis
- `instituional recognition.ipynb` – script for H2 institutional recognition coding and analysis

## Requirements
The following Python libraries are used:
- pandas
- biopython

## How to Run
Open each notebook in Jupyter Notebook or Google Colab and run the cells in order.
